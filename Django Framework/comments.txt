🔹Static Files
Static files = files that do NOT change
These are used for design and frontend styling.
Examples:
CSS files (styles)
JavaScript files
Images (logo, icons)
Fonts
📌 Key idea:
They are fixed and written by the developer.

🔹 Media Files
Media files = files that ARE uploaded by users
These are dynamic, meaning they come from users.
Examples:
Profile pictures
Uploaded documents
Videos uploaded by users
📌 Key idea:
They are not fixed — users create/upload them.
